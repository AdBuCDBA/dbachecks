# dbachecks

When you install, it'll auto-install:

* dbatools
* Pester
* PSFramework

When you import, it'll auto-import

* dbatools
* Pester
* PSFramework

## Getting started

```
Set-DbcConfig -Name SqlInstance -Value sql2016, sql2017, sqlcluster
Get-DbcConfig
Invoke-DbcCheck
```
